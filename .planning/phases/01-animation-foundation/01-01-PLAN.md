# Plan 01: Project Scaffold + Game Logic Contracts

## Objective

Scaffold the full SvelteKit project with Tailwind v4 and GSAP installed, SPA mode enabled,
and all pure-logic game library modules written and unit-tested. Wave 2 can import from
`$lib/game/` and `$lib/session/` immediately without exploring the codebase.

This plan establishes the type contracts and pure-logic modules that the SVG wheel component
(Wave 2) and localStorage integration (Wave 3) depend on. No component or animation code is
written here — only the foundation that makes those safe to implement.

## Phase Goal

**As a** developer, **I want to** see a single animated SVG wheel spin with correct
`power4.out` deceleration, land on a pre-determined result, and recover its state from
localStorage after a page reload, **so that** the animation architecture is validated before
all 23 wheels are built on top of it.

--- src/lib/session/types.ts ---
Export these interfaces (they are the contracts Wave 2 and Wave 3 import):
  - WeightedSegment: { label: string; weight: number; color?: string }
  - SpinStatus: type union 'IDLE' | 'SPINNING' | 'LANDED' | 'REVEALED'
  - SpinResult: { step: number; category: string; resultLabel: string; resultIndex: number; timestamp: string }
  - SessionState: { sessionId: string; startedAt: string; completedSpins: SpinResult[] }
No implementations — types only.

--- src/lib/session/store.ts ---
Import SessionState from './types'. Export these four functions:
  - createSession(): SessionState — generates sessionId via crypto.randomUUID(), sets
    startedAt to new Date().toISOString(), completedSpins to [].
  - loadSession(): SessionState | null — guards typeof localStorage === 'undefined',
    calls localStorage.getItem('wof_session'), parses JSON in try/catch returning null on
    any error (corrupted data = no session).
  - saveSession(session: SessionState): void — guards typeof localStorage, calls
    localStorage.setItem('wof_session', JSON.stringify(session)).
  - clearSession(): void — guards typeof localStorage, calls
    localStorage.removeItem('wof_session').
Storage key constant: STORAGE_KEY = 'wof_session' (defined once, used throughout).

--- src/lib/game/geometry.ts ---
Export these three functions (exact signatures below — Wave 2 imports them):

  slicePath(cx: number, cy: number, r: number, startDeg: number, endDeg: number): string
  - Converts startDeg and endDeg to radians using (deg - 90) * Math.PI / 180.
    The -90 offset shifts SVG's 0° (east/3 o'clock) to 0° at north/12 o'clock.
  - Computes arc endpoint coordinates using cos/sin.
  - large-arc-flag is 1 if (endDeg - startDeg) > 180, else 0.
  - Returns: M {cx} {cy} L {x1} {y1} A {r} {r} 0 {largeArc} 1 {x2} {y2} Z

  equalSegmentAngles(count: number): Array<{ startDeg: number; endDeg: number; midDeg: number }>
  - Returns one entry per segment; size = 360 / count.
  - startDeg = i * size, endDeg = (i+1) * size, midDeg = i * size + size / 2.

  calculateTargetAngle(currentRotation: number, targetIndex: number, totalSegments: number, minSpins?: number): number
  - minSpins defaults to 5.
  - segmentSize = 360 / totalSegments
  - segmentCenter = targetIndex * segmentSize + segmentSize / 2
  - currentMod = currentRotation % 360
  - delta = ((360 - segmentCenter) - currentMod + 360) % 360
  - Returns currentRotation + (minSpins * 360) + delta
  - NEVER resets currentRotation to 0 — this function always returns a value greater than
    currentRotation.

--- src/lib/game/random.ts ---
Export WeightedItem interface: { label: string; weight: number }.
Export weightedRandom(items: WeightedItem[]): number.
  - Sums total weight. Rolls Math.random() * total.
  - Iterates subtracting each weight; returns index when roll <= 0.
  - Falls back to items.length - 1 for floating-point edge case.
This function is called BEFORE any GSAP tween is created — result-before-animation.

--- src/lib/game/spinEngine.ts ---
Export SpinStatus re-exported from '../session/types'.
Export isValidTransition(from: SpinStatus, to: SpinStatus): boolean.
  Valid transitions only:
    IDLE -> SPINNING
    SPINNING -> LANDED
    LANDED -> REVEALED
    REVEALED -> IDLE
  All other transitions return false.
