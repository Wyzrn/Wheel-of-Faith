# Plan 03: localStorage Session Persistence + Resume Prompt

## Objective

Wire localStorage read/write into the page layer so users can resume a session after a
page reload. On load: check for an existing session and show a resume prompt if spins exist.
On spin complete: write the updated session to localStorage. Resume: restore last result
to the UI. Start Over: clear storage and reset.

This plan modifies only +page.svelte. The session store functions (loadSession, saveSession,
clearSession, createSession) are already implemented in Wave 1. SpinWheel.svelte already
calls onSpinComplete on each spin. This plan wires those pieces together.

  import { loadSession, saveSession, clearSession, createSession } from '$lib/session/store'

New reactive state:
  let currentSession = $state<SessionState>(createSession())
  let showResumePrompt = $state(false)
  let restoredResult = $state<{ index: number; label: string } | null>(null)

onMount block:
  const saved = loadSession()
  if (saved && saved.completedSpins.length > 0) {
    currentSession = saved
    showResumePrompt = true
  }
  The typeof guard in loadSession() handles the SSR edge case but since ssr=false is set
  at root layout level, this is belt-and-suspenders. Do NOT read localStorage directly here
  — always go through the store functions.

Update handleSpinComplete to persist after each completed spin:
  function handleSpinComplete(index: number, label: string) {
    const newSpin = {
      step: currentSession.completedSpins.length + 1,
      category: 'demo',
      resultLabel: label,
      resultIndex: index,
      timestamp: new Date().toISOString()
    }
    const updatedSession = {
      ...currentSession,
      completedSpins: [...currentSession.completedSpins, newSpin]
    }
    currentSession = updatedSession
    saveSession(updatedSession)
    spinHistory = [...spinHistory, { index, label }]
  }
  Write to localStorage AFTER recording the result — result-before-storage, matching the
  result-before-animation principle (CLAUDE.md rule #2).

Resume handler:
  function handleResume() {
    showResumePrompt = false
    const lastSpin = currentSession.completedSpins.at(-1)
    if (lastSpin) {
      restoredResult = { index: lastSpin.resultIndex, label: lastSpin.resultLabel }
      spinHistory = currentSession.completedSpins.map(s => ({
        index: s.resultIndex,
        label: s.resultLabel
      }))
    }
  }

Start Over handler:
  function handleStartOver() {
    clearSession()
    currentSession = createSession()
    showResumePrompt = false
    restoredResult = null
    spinHistory = []
  }

| T-01-03 | Tampering | SVG segment labels rendered as text | mitigate | All segment labels rendered via Svelte text interpolation ({label}), never via {@html}. Svelte auto-escapes — XSS via segment label content is not possible. |
