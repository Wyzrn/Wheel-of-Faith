# Plan 02: SVG Wheel Component + GSAP Animation

## Objective

Build the SpinWheel.svelte component: SVG pie wheel with N segments, a fixed pointer at
12 o'clock, GSAP power4.out rotation tween, and a four-state machine (IDLE → SPINNING →
LANDED → REVEALED). The result is determined before the animation starts. The pointer
position determines what the user sees — GSAP's job is only to move the wheel to the
pre-computed angle.

This plan depends on 01-01 (types, geometry, random, store modules must exist). It does NOT
write to localStorage — that is Wave 3.

Script block (lang="ts"):

Imports:
  import { onMount } from 'svelte'
  import { gsap } from 'gsap'
  import { slicePath, equalSegmentAngles, calculateTargetAngle } from '$lib/game/geometry'
  import { weightedRandom } from '$lib/game/random'
  import type { WeightedSegment, SpinStatus } from '$lib/session/types'

Props (via $props()):
  segments: WeightedSegment[] — receives the demo segment data from +page.svelte.
  onSpinComplete: (resultIndex: number, resultLabel: string) => void — callback Wave 3 will use.

Constants:
  SVG_SIZE = 400
  SVG_CENTER = '200 200'   — string literal, used in svgOrigin. Never use "50% 50%".
  WHEEL_RADIUS = 180
  POINTER_SIZE = 20

Reactive state (all $state):
  spinStatus = $state<SpinStatus>('IDLE')
  currentRotation = $state(0)  — accumulated, NEVER reset to 0
  lastResult = $state<{ index: number; label: string } | null>(null)

Derived:
  canSpin = $derived(spinStatus === 'IDLE')
  isRevealed = $derived(spinStatus === 'REVEALED')
  segmentAngles = $derived(equalSegmentAngles(segments.length))

Template variable (not reactive state):
  let wheelGroupEl: SVGGElement  — for bind:this, used by GSAP

Lifecycle:
  onMount receives a ref to wheelGroupEl via bind:this. Initialize gsap.context scoped
  to wheelGroupEl. Return () => ctx.revert() as the cleanup. Do NOT start any tween here
  (the wheel starts static). This is the onMount pattern — NOT $effect.

handleSpin() function (called by button click):
  Step 1: Guard — if spinStatus is not 'IDLE' return immediately (prevents double-spin).
  Step 2: weightedRandom(segments) — determines resultIndex BEFORE any animation.
  Step 3: calculateTargetAngle(currentRotation, resultIndex, segments.length, 5) — computes targetAngle.
  Step 4: Set spinStatus = 'SPINNING'.
  Step 5: gsap.to(wheelGroupEl, {
    rotation: targetAngle,
    duration: 4,
    ease: 'power4.out',
    svgOrigin: SVG_CENTER,
    force3D: true,
    onComplete: () => {
      currentRotation = targetAngle;   — preserve for next spin's delta
      lastResult = { index: resultIndex, label: segments[resultIndex].label };
      spinStatus = 'LANDED';
      onSpinComplete(resultIndex, segments[resultIndex].label);
      setTimeout(() => { spinStatus = 'REVEALED'; }, 500);
    }
  });
  The onComplete callback is the ONLY place the result is read. Never call
  gsap.getProperty(wheelGroupEl, 'rotation') to determine which segment the wheel stopped on.

Template (HTML/SVG):
  Outer wrapper div: Tailwind classes for centering (flex flex-col items-center gap-6).

  SVG element: width={SVG_SIZE} height={SVG_SIZE} viewBox="0 0 {SVG_SIZE} {SVG_SIZE}"
    Add aria-label="Spinning wheel" role="img".

  Inside SVG, two children:
  1. <g bind:this={wheelGroupEl}> — this is the element GSAP rotates.
     Inside: one <path> per segment using slicePath(200, 200, WHEEL_RADIUS, startDeg, endDeg).
     Fill: use segment.color if provided, else cycle through a color palette
     (e.g. ['#E63946','#457B9D','#2A9D8F','#E9C46A','#F4A261','#264653','#6A0572','#0077B6']).
     Stroke: white, stroke-width 2.
     Each slice gets a <text> label at the segment midpoint:
       x = 200 + (WHEEL_RADIUS * 0.65) * cos((midDeg - 90) * PI / 180)
       y = 200 + (WHEEL_RADIUS * 0.65) * sin((midDeg - 90) * PI / 180)
     text-anchor="middle", dominant-baseline="middle", fill="white", font-size="13".
     Use the raw SVG text element — do NOT use {@html} — Svelte escapes segment.label
     automatically preventing XSS.
     Winning segment highlight: when isRevealed and lastResult?.index === i, apply
     stroke="gold" stroke-width="4" to that path. Implement with a conditional attribute.

  2. Fixed pointer outside the <g>: a triangle <polygon> pointing downward at the top center
     of the wheel (12 o'clock position). Coordinates: roughly
     points="200,{200 - WHEEL_RADIUS - 5} {200 - POINTER_SIZE/2},{200 - WHEEL_RADIUS - POINTER_SIZE - 5} {200 + POINTER_SIZE/2},{200 - WHEEL_RADIUS - POINTER_SIZE - 5}"
     fill="#FF6B35" (orange-red). This sits OUTSIDE the <g> so it does NOT rotate with the wheel.

  Below the SVG:
    - Spin button: onclick={handleSpin} disabled={!canSpin}
      Text: spinStatus === 'IDLE' ? 'Spin' : spinStatus
      Tailwind classes: bg-indigo-600 text-white px-8 py-3 rounded-lg text-lg
        disabled:opacity-40 disabled:cursor-not-allowed hover:bg-indigo-700 transition-colors
    - Result display (only when isRevealed):
      {#if isRevealed && lastResult}
        <p class="text-2xl font-bold text-center">Result: {lastResult.label}</p>
      {/if}

Do NOT wrap the SVG in a fixed-size container that clips on mobile. Use max-w-sm mx-auto
with w-full on the SVG's wrapper div so it scales with viewport width.
